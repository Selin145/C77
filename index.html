Visual Studio Code (VS Code) is a lightweight and versatile source code editor developed by Microsoft. Set it up by downloading the installer from the official website and customizing it with extensions and settings. Open projects or files to start coding with features like syntax highlighting and autocompletion. The console screen, or terminal, is a text-based interface to interact with the operating system. Open it using the Run dialog on Windows or the Terminal application on macOS/Linux. Execute commands and perform tasks. To print on the console, use commands specific to the terminal:

Windows Command Prompt: Use "echo" (e.g., echo Hello, World!). PowerShell: Use "Write-Host" (e.g., Write-Host "Hello, World!"). macOS/Linux Terminal: Use "echo" (e.g., echo Hello, World!). VS Code offers a rich ecosystem of extensions and tools, including an integrated terminal, debugging, version control, and more. It provides a customizable coding environment, while the console screen allows direct interaction with the operating system.
